# API Docs (docs.rs)

Published naia crates:

- [`naia-client`](https://docs.rs/naia-client) — core client API
- [`naia-server`](https://docs.rs/naia-server) — core server API
- [`naia-shared`](https://docs.rs/naia-shared) — protocol, channels, replication primitives
- [`naia-derive`](https://docs.rs/naia-derive) — derive macros re-exported by shared crates
- [`naia-serde`](https://docs.rs/naia-serde) — bit-level serialization primitives
- [`naia-serde-derive`](https://docs.rs/naia-serde-derive) — serde derive macros
- [`naia-client-socket`](https://docs.rs/naia-client-socket) — lower-level client socket support
- [`naia-server-socket`](https://docs.rs/naia-server-socket) — lower-level server socket support
- [`naia-socket-shared`](https://docs.rs/naia-socket-shared) — shared socket types/configuration
- [`naia-bevy-client`](https://docs.rs/naia-bevy-client) — Bevy client adapter
- [`naia-bevy-server`](https://docs.rs/naia-bevy-server) — Bevy server adapter
- [`naia-bevy-shared`](https://docs.rs/naia-bevy-shared) — Bevy shared protocol/component helpers
- [`naia-metrics`](https://docs.rs/naia-metrics) — optional metrics primitives
- [`naia-bevy-metrics`](https://docs.rs/naia-bevy-metrics) — optional Bevy metrics integration

This book explains concepts and patterns. docs.rs is the source for exact type
signatures, trait bounds, feature flags, and rustdoc examples.
