# API Docs (docs.rs)

The auto-generated API documentation for all naia crates is available on docs.rs:

- [`naia-shared`](https://docs.rs/naia-shared) — protocol definition, derives, channel types
- [`naia-server`](https://docs.rs/naia-server) — core server API
- [`naia-client`](https://docs.rs/naia-client) — core client API
- [`naia-bevy-server`](https://docs.rs/naia-bevy-server) — Bevy server adapter
- [`naia-bevy-client`](https://docs.rs/naia-bevy-client) — Bevy client adapter
- [`naia-macroquad-client`](https://docs.rs/naia-macroquad-client) — macroquad adapter

---

The docs.rs pages include full rustdoc output with type signatures, trait bounds,
and inline examples. This book focuses on conceptual explanations and patterns;
the API docs are the reference for specific method signatures and return types.
