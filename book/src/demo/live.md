# Try It In Your Browser

> **Coming soon.** An in-browser demo is planned — a full naia session running
> entirely in WASM via `transport_local`, no backend required.

In the meantime, clone the repo and run any of the included demos locally:

```sh
# Bevy demo (native)
cargo run -p naia-bevy-server-demo    # terminal 1
cargo run -p naia-bevy-client-demo    # terminal 2
```

See [Running the Demos](../getting-started/demos.md) for the full list.
